## Run Scripts

Dev Mode - `npm start`
Production Mode - `npm run build`

### Filter Movies (GET)
 Query - `genre=drama,year=1990`


### Rank Movies (Rank)
Query - `title='Knives Out'&rank=1&title='They Shall Not Grow Old'&rank=2`
